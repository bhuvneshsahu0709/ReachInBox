# ReachInBox

Minimal onebox-style email dashboard with a Node/Express backend and React/Vite frontend.

## Prerequisites
- Node.js 18+
- npm 9+

## Install
```bash
cd backend
npm install

cd ../frontend
npm install
```

## Run (dummy mode)
Backend (API):
```bash
cd backend
npm run dev
```

Frontend (UI):
```bash
cd frontend
npm run dev
```

Then open `http://localhost:5173` in your browser.
